enum ASGE::SpriteSortMode
strong

Used to control the sprite batching techniques that renderers may support.

Enumerator
IMMEDIATE 

Renders the sprite immediately.

DEFERRED 

Defers (batches) the rendering to end of the render cycle.

TEXTURE 

Batches and sorts by texture ID.

BACK_TO_FRONT 

Batches and reverses the order using z-ordering.

FRONT_TO_BACK 

Batches and sorts the order using z-ordering.
